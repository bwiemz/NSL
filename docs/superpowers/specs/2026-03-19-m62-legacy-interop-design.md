# M62: Legacy Interop — Zero-Copy PyTorch/JAX FFI — Design Spec

**Date:** 2026-03-19
**Status:** Planned
**Milestone:** M62
**Prerequisites:** M18 (Interop — existing py.call(), from_torch(), ONNX export)
**Dependencies:** Unblocks adoption of all subsequent milestones (M58-M61 require labs using NSL)

---

## Overview

Allow researchers to call NSL-compiled models from Python, and call Python functions from NSL, with zero-copy tensor sharing via DLPack. This is the adoption gate — no frontier lab will rewrite their entire evaluation harness, data pipeline, and experiment tracking infrastructure in a new language. NSL must integrate seamlessly with existing PyTorch/JAX codebases.

**Key design:** `nsl build --shared-lib` compiles an NSL model into a shared library (`.so`/`.dylib`/`.dll`) with a stable C API. A Python package (`pip install nslpy`) wraps this C API with ergonomic Python bindings that support zero-copy tensor exchange via DLPack, bidirectional function calls, gradient bridging with `torch.autograd`, HuggingFace `from_pretrained()` integration, and ONNX Runtime custom op registration.

**Why this is critical for adoption:** The adoption path for a new ML language is incremental. Researchers will not port their entire codebase on day one. They need to: (1) load a pretrained model into NSL for fast inference, (2) call the NSL model from their existing Python evaluation code, (3) gradually migrate layers to NSL while keeping the rest in PyTorch, (4) eventually run the full pipeline in NSL. Each step must work with zero friction.

**Why DLPack:** DLPack is the industry-standard zero-copy tensor interchange format, supported by PyTorch, JAX, TensorFlow, CuPy, and NumPy (via `__dlpack__` protocol). It defines a C struct (`DLManagedTensor`) that describes tensor memory layout without copying data. NSL's `NslTensor` can be wrapped in a `DLManagedTensor` for export, or an incoming `DLManagedTensor` can be wrapped in an `NslTensor` for import — in both cases, the underlying memory is shared, not copied.

## Section 1: Language Surface

### CLI: Building Shared Libraries

```
# Compile model to shared library with C API
nsl build --shared-lib model.nsl -o libmodel.so

# Compile with specific exports (only expose certain functions)
nsl build --shared-lib --export "forward,generate" model.nsl

# Compile with weights baked in (uses M52 weight-aware compilation)
nsl build --shared-lib --weights model.safetensors model.nsl -o libmodel_optimized.so

# Compile for multiple platforms
nsl build --shared-lib --target linux-x86_64 model.nsl -o libmodel.so
nsl build --shared-lib --target darwin-arm64 model.nsl -o libmodel.dylib
nsl build --shared-lib --target windows-x86_64 model.nsl -o model.dll
```

### Python Usage: `nslpy`

```python
import torch
import nslpy

# Load an NSL-compiled shared library
model = nslpy.load("libmodel.so")

# Zero-copy tensor exchange: PyTorch tensor → NSL
input_tensor = torch.randn(1, 512, dtype=torch.float32, device="cuda:0")
output = model.forward(input_tensor)  # output is a torch.Tensor (zero-copy back)

# Automatic device matching
cpu_input = torch.randn(1, 512)
cpu_output = model.forward(cpu_input)  # runs on CPU if input is CPU

# Multiple outputs
logits, kv_cache = model.forward_with_cache(input_tensor, past_kv=None)

# HuggingFace integration
model = nslpy.from_pretrained("meta-llama/Llama-3-70B", lib="libllama.so")
output = model.generate("Hello, world!", max_tokens=100)

# Gradient bridging: NSL backward compatible with torch.autograd
input_tensor = torch.randn(1, 512, requires_grad=True, device="cuda:0")
output = model.forward(input_tensor)  # returns a torch.Tensor with grad_fn
loss = output.sum()
loss.backward()  # gradients flow back through NSL model
print(input_tensor.grad)  # valid gradient tensor
```

### NslTensorDesc to/from DLPack Conversion

```rust
/// Convert NslTensor (internal) to DLManagedTensor (DLPack standard).
pub fn nsl_tensor_to_dlpack(tensor: &NslTensor) -> *mut DLManagedTensor {
    let dl_tensor = DLTensor {
        data: tensor.data,
        device: DLDevice {
            device_type: if tensor.device == 0 {
                DLDeviceType::kDLCPU
            } else {
                DLDeviceType::kDLCUDA
            },
            device_id: if tensor.device == 0 { 0 } else { (tensor.device - 1) as i32 },
        },
        ndim: tensor.ndim as i32,
        dtype: nsl_dtype_to_dl_dtype(tensor.dtype),
        shape: tensor.shape.as_ptr() as *mut i64,
        strides: std::ptr::null_mut(), // contiguous layout
        byte_offset: 0,
    };

    let managed = Box::new(DLManagedTensor {
        dl_tensor,
        manager_ctx: std::ptr::null_mut(),
        deleter: Some(nsl_dlpack_deleter),
    });

    Box::into_raw(managed)
}

/// Convert DLManagedTensor (from PyTorch/JAX) to NslTensor (internal).
/// Zero-copy: wraps the existing memory, does not copy data.
pub fn dlpack_to_nsl_tensor(dl: *const DLManagedTensor) -> NslTensor {
    let dl = unsafe { &*dl };
    let t = &dl.dl_tensor;

    let device = match t.device.device_type {
        DLDeviceType::kDLCPU => 0u8,
        DLDeviceType::kDLCUDA => (t.device.device_id + 1) as u8,
        _ => panic!("Unsupported DLPack device type"),
    };

    let dtype = dl_dtype_to_nsl_dtype(&t.dtype);

    let shape: Vec<u64> = (0..t.ndim as usize)
        .map(|i| unsafe { *t.shape.add(i) } as u64)
        .collect();

    let numel: u64 = shape.iter().product();

    NslTensor {
        data: t.data,
        ndim: t.ndim as u32,
        shape: shape.as_ptr() as *const u64,
        numel,
        device,
        dtype,
        // ... other fields ...
    }
}
```

### DLPack Type Mappings

```rust
fn nsl_dtype_to_dl_dtype(nsl_dtype: u8) -> DLDataType {
    match nsl_dtype {
        0 => DLDataType { code: DLDataTypeCode::kDLFloat, bits: 32, lanes: 1 },   // f32
        1 => DLDataType { code: DLDataTypeCode::kDLFloat, bits: 64, lanes: 1 },   // f64
        2 => DLDataType { code: DLDataTypeCode::kDLFloat, bits: 16, lanes: 1 },   // f16
        3 => DLDataType { code: DLDataTypeCode::kDLBfloat, bits: 16, lanes: 1 },  // bf16
        4 => DLDataType { code: DLDataTypeCode::kDLInt, bits: 32, lanes: 1 },     // int32
        5 => DLDataType { code: DLDataTypeCode::kDLInt, bits: 64, lanes: 1 },     // int64
        6 => DLDataType { code: DLDataTypeCode::kDLInt, bits: 8, lanes: 1 },      // int8
        7 => DLDataType { code: DLDataTypeCode::kDLUInt, bits: 8, lanes: 1 },     // uint8
        8 => DLDataType { code: DLDataTypeCode::kDLFloat, bits: 8, lanes: 1 },    // fp8 (e4m3)
        _ => panic!("Unknown NSL dtype: {}", nsl_dtype),
    }
}

fn dl_dtype_to_nsl_dtype(dl: &DLDataType) -> u8 {
    match (dl.code, dl.bits) {
        (DLDataTypeCode::kDLFloat, 32) => 0,
        (DLDataTypeCode::kDLFloat, 64) => 1,
        (DLDataTypeCode::kDLFloat, 16) => 2,
        (DLDataTypeCode::kDLBfloat, 16) => 3,
        (DLDataTypeCode::kDLInt, 32) => 4,
        (DLDataTypeCode::kDLInt, 64) => 5,
        (DLDataTypeCode::kDLInt, 8) => 6,
        (DLDataTypeCode::kDLUInt, 8) => 7,
        (DLDataTypeCode::kDLFloat, 8) => 8,
        _ => panic!("Unsupported DLPack dtype: {:?}/{}", dl.code, dl.bits),
    }
}
```

### Memory Management: Ownership Protocol

```rust
/// When NSL exports a tensor to Python via DLPack:
/// 1. NSL allocates the DLManagedTensor wrapper (Box::new)
/// 2. The dl_tensor.data points to NSL's existing tensor memory
/// 3. Python calls the deleter when it's done with the DLManagedTensor
/// 4. The deleter frees the wrapper but NOT the underlying data
///    (NSL still owns the tensor data)
unsafe extern "C" fn nsl_dlpack_deleter(managed: *mut DLManagedTensor) {
    if !managed.is_null() {
        // Free the shape array we allocated
        let m = &*managed;
        if !m.dl_tensor.shape.is_null() {
            let _ = Vec::from_raw_parts(
                m.dl_tensor.shape,
                m.dl_tensor.ndim as usize,
                m.dl_tensor.ndim as usize,
            );
        }
        // Free the DLManagedTensor struct itself
        let _ = Box::from_raw(managed);
        // NOTE: We do NOT free dl_tensor.data — NSL still owns it
    }
}

/// When Python exports a tensor to NSL via DLPack:
/// 1. Python creates a DLManagedTensor via torch.Tensor.__dlpack__()
/// 2. NSL wraps the dl_tensor in an NslTensor (zero-copy)
/// 3. When NSL is done, it calls the original deleter
/// 4. The deleter notifies PyTorch to release its reference
pub struct BorrowedDLTensor {
    managed: *mut DLManagedTensor,
    nsl_tensor: NslTensor,
}

impl Drop for BorrowedDLTensor {
    fn drop(&mut self) {
        unsafe {
            if let Some(deleter) = (*self.managed).deleter {
                deleter(self.managed);
            }
        }
    }
}
```

```python
# nslpy/model.py
import ctypes
import torch
import numpy as np
from ._core import _lib, NslTensorDesc, DLManagedTensorPtr

class NslModel:
    """Wrapper around an NSL-compiled shared library."""

    def __init__(self, lib_path: str, weights_path: str | None = None):
        self._handle = _lib.nsl_model_create(
            weights_path.encode() if weights_path else None
        )
        if not self._handle:
            raise RuntimeError(f"Failed to load NSL model: {_lib.nsl_get_last_error()}")

        # Parse metadata to discover exported functions
        meta_json = _lib.nsl_model_get_metadata(self._handle)
        self._metadata = json.loads(meta_json)
        self._exports = self._metadata.get("exports", {})

    def forward(self, *inputs: torch.Tensor) -> torch.Tensor | tuple[torch.Tensor, ...]:
        """Call the model's forward function with zero-copy DLPack tensors."""
        return self._call_dlpack("forward", inputs)

    def __call__(self, *inputs: torch.Tensor) -> torch.Tensor | tuple[torch.Tensor, ...]:
        return self.forward(*inputs)

    def _call_dlpack(self, func_name: str, inputs: tuple[torch.Tensor, ...]):
        """Zero-copy function call via DLPack."""
        # Convert PyTorch tensors to DLPack capsules
        dl_inputs = []
        for t in inputs:
            dl_capsule = torch.utils.dlpack.to_dlpack(t)
            dl_inputs.append(dl_capsule)

        # Call NSL function
        dl_inputs_arr = (DLManagedTensorPtr * len(dl_inputs))(*dl_inputs)
        num_outputs = ctypes.c_int(0)
        dl_outputs = ctypes.POINTER(DLManagedTensorPtr)()

        status = _lib.nsl_model_forward_dlpack(
            self._handle,
            dl_inputs_arr,
            len(dl_inputs),
            ctypes.byref(dl_outputs),
            ctypes.byref(num_outputs),
        )

        if status != 0:
            raise RuntimeError(f"NSL forward failed: {_lib.nsl_get_last_error()}")

        # Convert DLPack outputs back to PyTorch tensors (zero-copy)
        results = []
        for i in range(num_outputs.value):
            torch_tensor = torch.utils.dlpack.from_dlpack(dl_outputs[i])
            results.append(torch_tensor)

        if len(results) == 1:
            return results[0]
        return tuple(results)

    def call(self, func_name: str, *inputs: torch.Tensor):
        """Call a named exported function."""
        return self._call_dlpack(func_name, inputs)

    def __del__(self):
        if hasattr(self, '_handle') and self._handle:
            _lib.nsl_model_destroy(self._handle)

    @property
    def num_parameters(self) -> int:
        return _lib.nsl_model_get_num_parameters(self._handle)

    @property
    def metadata(self) -> dict:
        return self._metadata
```

### Gradient Bridge: torch.autograd Integration

```python
# nslpy/autograd.py
import torch
from torch.autograd import Function

class NslFunction(Function):
    """torch.autograd.Function that bridges NSL forward/backward."""

    @staticmethod
    def forward(ctx, model_handle, *inputs):
        """Forward pass through NSL model, recording context for backward."""
        dl_inputs = [torch.utils.dlpack.to_dlpack(t) for t in inputs]
        dl_inputs_arr = (DLManagedTensorPtr * len(dl_inputs))(*dl_inputs)

        num_outputs = ctypes.c_int(0)
        dl_outputs = ctypes.POINTER(DLManagedTensorPtr)()
        grad_ctx = ctypes.c_void_p()

        status = _lib.nsl_model_forward_grad(
            model_handle,
            dl_inputs_arr,
            len(dl_inputs),
            ctypes.byref(dl_outputs),
            ctypes.byref(num_outputs),
            ctypes.byref(grad_ctx),
        )

        if status != 0:
            raise RuntimeError(f"NSL forward_grad failed: {_lib.nsl_get_last_error()}")

        # Save context for backward
        ctx.nsl_grad_ctx = grad_ctx
        ctx.num_inputs = len(inputs)

        # Convert outputs to torch tensors
        outputs = []
        for i in range(num_outputs.value):
            outputs.append(torch.utils.dlpack.from_dlpack(dl_outputs[i]))

        return tuple(outputs)

    @staticmethod
    def backward(ctx, *grad_outputs):
        """Backward pass: compute gradients w.r.t. inputs using NSL's autodiff."""
        dl_grad_outputs = [torch.utils.dlpack.to_dlpack(g) for g in grad_outputs]
        dl_grad_arr = (DLManagedTensorPtr * len(dl_grad_outputs))(*dl_grad_outputs)

        num_grad_inputs = ctypes.c_int(0)
        dl_grad_inputs = ctypes.POINTER(DLManagedTensorPtr)()

        status = _lib.nsl_model_backward(
            ctx.nsl_grad_ctx,
            dl_grad_arr,
            len(dl_grad_outputs),
            ctypes.byref(dl_grad_inputs),
            ctypes.byref(num_grad_inputs),
        )

        if status != 0:
            raise RuntimeError(f"NSL backward failed: {_lib.nsl_get_last_error()}")

        # Convert gradient DLPack tensors to torch
        grad_inputs = [None]  # first arg is model_handle, no grad
        for i in range(num_grad_inputs.value):
            grad_inputs.append(torch.utils.dlpack.from_dlpack(dl_grad_inputs[i]))

        # Free the NSL grad context
        _lib.nsl_model_free_grad_context(ctx.nsl_grad_ctx)

        return tuple(grad_inputs)


class NslGradModel:
    """Model wrapper that supports torch.autograd gradient flow."""

    def __init__(self, model: 'NslModel'):
        self._model = model

    def __call__(self, *inputs: torch.Tensor) -> torch.Tensor | tuple[torch.Tensor, ...]:
        outputs = NslFunction.apply(self._model._handle, *inputs)
        if len(outputs) == 1:
            return outputs[0]
        return outputs
```

## Section 5: HuggingFace Integration

```python
# nslpy/huggingface.py
from huggingface_hub import hf_hub_download, model_info
from pathlib import Path

def from_pretrained(
    model_id: str,
    lib_path: str | None = None,
    revision: str = "main",
    cache_dir: str | None = None,
    device: str = "cuda:0",
) -> 'NslModel':
    """Load a pretrained model from HuggingFace Hub into an NSL-compiled binary.

    Usage:
        model = nslpy.from_pretrained("meta-llama/Llama-3-70B", lib="libllama.so")
        output = model.generate(input_ids)
    """
    # 1. Download safetensors weights from HuggingFace
    info = model_info(model_id, revision=revision)
    safetensor_files = [
        f.rfilename for f in info.siblings
        if f.rfilename.endswith('.safetensors')
    ]

    weights_dir = Path(cache_dir or Path.home() / ".cache" / "nslpy" / model_id)
    weights_dir.mkdir(parents=True, exist_ok=True)

    for filename in safetensor_files:
        local_path = weights_dir / filename
        if not local_path.exists():
            hf_hub_download(
                model_id,
                filename=filename,
                revision=revision,
                local_dir=str(weights_dir),
            )

    # 2. If lib_path is provided, load the NSL-compiled library
    if lib_path:
        model = NslModel(lib_path, weights_path=str(weights_dir))
        return model

    # 3. If no lib_path, check for a pre-compiled NSL binary in the HF repo
    nsl_lib_file = next(
        (f.rfilename for f in info.siblings if f.rfilename.endswith('.so')),
        None
    )

    if nsl_lib_file:
        lib_local = hf_hub_download(model_id, nsl_lib_file, local_dir=str(weights_dir))
        return NslModel(lib_local, weights_path=str(weights_dir))

    raise ValueError(
        f"No NSL shared library found for {model_id}. "
        f"Provide lib_path to a pre-compiled .so file."
    )
```

### Weight Loading Protocol

```rust
/// Load safetensors weights into the model at runtime.
/// Called by nsl_model_create(weights_path).
pub fn load_weights_from_safetensors(
    model: &mut NslModelInstance,
    path: &Path,
) -> Result<(), WeightLoadError> {
    // Handle single file or sharded (model-00001-of-00004.safetensors)
    let files = if path.is_dir() {
        // Sharded: load all safetensor files in directory
        let mut files: Vec<PathBuf> = std::fs::read_dir(path)?
            .filter_map(|e| e.ok())
            .map(|e| e.path())
            .filter(|p| p.extension().map_or(false, |e| e == "safetensors"))
            .collect();
        files.sort();
        files
    } else {
        vec![path.to_path_buf()]
    };

    for file in &files {
        let safetensors = safetensors::SafeTensors::deserialize(
            &std::fs::read(file)?
        )?;

        for (name, tensor_view) in safetensors.tensors() {
            // Map safetensors tensor name to NSL model parameter
            if let Some(param) = model.find_parameter(&name) {
                // Verify shape matches
                if tensor_view.shape() != param.shape() {
                    return Err(WeightLoadError::ShapeMismatch {
                        name: name.to_string(),
                        expected: param.shape().to_vec(),
                        got: tensor_view.shape().to_vec(),
                    });
                }

                // Copy weight data into model parameter (with dtype conversion if needed)
                let src_dtype = safetensors_dtype_to_nsl(tensor_view.dtype());
                if src_dtype == param.dtype() {
                    // Same dtype: direct memcpy
                    unsafe {
                        std::ptr::copy_nonoverlapping(
                            tensor_view.data().as_ptr(),
                            param.data_mut() as *mut u8,
                            tensor_view.data().len(),
                        );
                    }
                } else {
                    // Different dtype: convert (e.g., bf16 safetensors → f32 model)
                    convert_and_copy(tensor_view.data(), src_dtype, param.data_mut(), param.dtype());
                }

                // If GPU model, transfer to GPU
                if param.device() > 0 {
                    param.to_device(param.device())?;
                }
            }
        }
    }

    Ok(())
}
```

```python
# nslpy/onnx_op.py
import onnxruntime as ort

def register_nsl_op(
    model: 'NslModel',
    op_name: str = "NslModelForward",
    domain: str = "com.nsl",
) -> None:
    """Register an NSL model as a custom ONNX Runtime operator.

    This allows NSL-compiled subgraphs to be embedded in ONNX models
```

```rust
pub struct SharedLibCompiler {
    exports: Vec<ExportedFunction>,
    weights_embedded: bool,
    target_platform: TargetPlatform,

pub struct ExportedFunction {
    pub nsl_name: String,
    pub c_name: String,
    pub input_types: Vec<TensorType>,
    pub output_types: Vec<TensorType>,
    pub supports_grad: bool,
}

pub enum TargetPlatform {
    LinuxX86_64,
    DarwinArm64,
    DarwinX86_64,
    WindowsX86_64,
}

impl SharedLibCompiler {
    /// Emit the C API wrapper functions for each @export function.
    pub fn emit_c_api_wrappers(&self, compiler: &mut Compiler) {
        for export in &self.exports {
            // Emit a C-callable wrapper that:
            // 1. Converts DLManagedTensor inputs to NslTensors (zero-copy)
            // 2. Calls the actual NSL function
            // 3. Converts NslTensor outputs to DLManagedTensors (zero-copy)
            self.emit_dlpack_wrapper(compiler, export);

            // Emit NslTensorDesc-based wrapper (for non-DLPack users)
            self.emit_tensordesc_wrapper(compiler, export);

            // If supports_grad, emit gradient-enabled wrapper
            if export.supports_grad {
                self.emit_grad_wrapper(compiler, export);
            }
        }

        // Emit model lifecycle functions
        self.emit_model_create(compiler);
        self.emit_model_destroy(compiler);
        self.emit_metadata(compiler);
    }

    /// Emit the DLPack wrapper for a single exported function.
    fn emit_dlpack_wrapper(&self, compiler: &mut Compiler, export: &ExportedFunction) {
        // Function signature:
        //   int nsl_<c_name>_dlpack(NslModel*, DLManagedTensor**, int, DLManagedTensor**, int*)

        // 2. Call the actual NSL function with NslTensor args
```

```rust
fn check_export_decorator(
    &mut self,
    func: &FnDef,
    decorator: &Decorator,
) -> Vec<Diagnostic> {
    let mut diags = Vec::new();

    // 1. @export functions must have tensor or primitive args only
    //    (no closures, no custom types in the C API)
    for param in &func.params {
        if !is_c_api_compatible_type(&param.ty) {
            diags.push(Diagnostic::error(
                format!("@export function parameter '{}' has type '{}' which is not C API compatible",
                    param.name, param.ty),
                param.span,
            ));
        }
    }

    // 2. Return type must be tensor or tuple of tensors
    if !is_c_api_compatible_return(&func.return_type) {
        diags.push(Diagnostic::error(
            "@export function return type must be a Tensor or tuple of Tensors",
            func.span,
        ));
    }

    // 3. @export functions cannot capture free variables
    if !func.captures.is_empty() {
        diags.push(Diagnostic::error(
            "@export function cannot capture variables from enclosing scope",
            func.span,
        ));
    }

    diags
}
```

## Section 9: Gradient Context

### Gradient Context for torch.autograd Bridge

When `nsl_model_forward_grad` is called, NSL runs its forward pass with autodiff tape recording enabled (M12). The tape context is returned as an opaque pointer that the Python side stores in `ctx.nsl_grad_ctx`. When backward is called, NSL replays the tape to compute input gradients.

```rust
pub struct GradContext {
    /// The autodiff tape from the forward pass
    pub tape: AutodiffTape,
    /// Input tensors (borrowed references for gradient computation)
    pub inputs: Vec<*const NslTensor>,
    /// Output tensors (for chain rule application)
    pub outputs: Vec<*mut NslTensor>,
}

impl GradContext {
    pub fn backward(
        &self,
        grad_outputs: &[&DLManagedTensor],
    ) -> Vec<*mut DLManagedTensor> {
        // 1. Convert grad_outputs from DLPack to NslTensor
        let nsl_grad_outputs: Vec<NslTensor> = grad_outputs.iter()
            .map(|dl| dlpack_to_nsl_tensor(*dl as *const _))
            .collect();

        // 2. Run NSL backward pass using the recorded tape
        let grad_inputs = self.tape.backward(&nsl_grad_outputs);

        // 3. Convert gradient NslTensors to DLPack for return to PyTorch
        grad_inputs.iter()
            .map(|t| nsl_tensor_to_dlpack(t))
            .collect()
    }
}
```
